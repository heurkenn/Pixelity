# Templates Iso

Fichiers fournis :

- `iso_tile_template.png`
  - gabarit de case au sol
  - taille : `96x48`

- `iso_building_frame_template.png`
  - gabarit d'une frame de batiment
  - taille : `96x96`

- `iso_building_sheet_template.png`
  - gabarit d'une feuille complete 3 frames
  - taille : `288x96`

Repères :

- rouge : empreinte isometrique utile
- bleu : axes de centrage
- vert : zone de confort generale pour le sprite

Conseil :

- garde le bas du batiment aligne sur le bas-centre du losange
- laisse le batiment monter vers le haut de la frame
- exporte en `PNG` avec fond transparent
